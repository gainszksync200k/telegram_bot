
Данный скрипт позволит вам создать собстенного телеграм бота, для отслеживания самых важных данных о рынке (СТОИМОСТЬ ЭФИРА,БИТКА,ГАЗ,ДОМИНАЦИЮ БИТКА,ИНДЕКС СТРАХА, ГАЗ В ЭФИРЕ) и балансе портфеля и получения их в чат телеги.

Настрока:

1. Пишем в консоль pip install -r requirements.txt  / для mac os pip3 install -r requirements.txt

Шаг 2
    : Создать бота через BotFather

    Откройте Telegram и найдите бота BotFather:

    В поиске введите @BotFather и выберите его из списка.

    Или перейдите по прямой ссылке: https://t.me/BotFather.

    Создайте нового бота:

    Напишите команду /start, чтобы начать взаимодействие с BotFather.

    Затем отправьте команду /newbot.

    BotFather попросит вас ввести имя бота (например, MyTestBot). Это имя будет отображаться в списке контактов.


    После ввода имени BotFather попросит вас выбрать уникальный username для бота. Он должен заканчиваться на bot (например, MyTestBot123_bot).

    Если выбранный username уже занят, BotFather попросит придумать другой.


    После успешного создания бота BotFather отправит вам сообщение с API-токеном. Он выглядит примерно так:

    123456789:ABCdefGhIJKlmNoPQRstuVWXyz

    Сохраните этот токен, он понадобится для управления ботом через код.!!!!!!!!!!!

Шаг 3

    ПОЛУЧАЕМ СВОЙ АЙДИ ТЕЛЕГИ


    Откройте Telegram Desktop (настольная версия Telegram).

    Перейдите в Настройки → Расширенные → Включить режим разработчика.


    После включения режима разработчика:

    Перейдите в свой профиль.

    Нажмите на три точки (⋮) рядом с вашим именем.

    Выберите Копировать ID аккаунта.

    Выглядит он примерно так 58991215
    Это ваш user_id.
